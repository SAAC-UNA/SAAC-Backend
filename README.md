SAAC Backend

Este proyecto es el backend de SAAC-UNA. Aquí se encuentra la lógica de negocio, las rutas de la API y la conexión con la base de datos.

Para desarrolladores:

-   PHP 8.2+
-   Laravel 12
-   Base de datos SQLite (por defecto)

Comandos útiles:

-   `php artisan migrate`
-   `php artisan db:seed`
-   `php artisan test`
